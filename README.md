# Project setup :

## Clone the repository :
```
git clone https://github.com/tapan0p/ScholarMind.git
cd ScholarMind
```

## setup webapp :
```
cd webapp
npm install
npm run dev
```

## setup backend :
```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
